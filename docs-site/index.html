Быстрый старт

Три ячейки — и ComfyUI работает

В Kaggle-блокноте, с включённым GPU T4 ×2 и интернетом.

# 0. Скрипты из репозитория (или обновить, если уже склонировано)
!git clone https://github.com/THE-ANGEL-AI/Kaggle_Workspace_FreeGPU.git || \
  git -C Kaggle_Workspace_FreeGPU pull

# 1. Окружение: uv + venv (Python 3.12) + torch cu130 + ComfyUI + Manager
!python Kaggle_Workspace_FreeGPU/instal/instal_comfyui.py

# 2. Кастомные ноды + симлинки на модели из /kaggle/input
!python Kaggle_Workspace_FreeGPU/instal/instal_castom_node.py

# 3. Запуск + Cloudflare-туннель + keep-alive кнопки
%run Kaggle_Workspace_FreeGPU/instal/start.py

Можно одной строкой: %run .../instal/start.py сам проверит окружение и вызовет нужный установщик автоматически — для холодного старта этого достаточно.
